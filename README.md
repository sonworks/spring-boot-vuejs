# spring-boot-vuejs

## Install vue webpack

```
$ vue init webpack frontend
```

* Build
```
$ cd frontend
$ npm run build
```

* Run DEV
```
$ cd frontend
$ npm run dev
```

## Install vuetify

```
$ vue create frontend-vuetify
$ cd frontend-vuetify/
$ vue add vuetify
$ npm install --save vue-router
$ npm install --save axios
$ npm install --save-dev eslint eslint-plugin-vue@next
```

* Build
```
$ cd frontend-vuetify
$ npm run build
```

* Run
```
$ cd frontend-vuetify
$ npm run serve
```
